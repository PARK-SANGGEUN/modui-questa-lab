# QUESTA OS 2022 Final

2022 개정 교육과정 과목명 기반 대학급·세부학과 맞춤 탐구 성장 운영체제입니다.

## 핵심 반영
- 2022 개정 과목군/과목 드롭다운
- 대학급 12단계 세분화
- 예체능 포함
- 계열별 세부학과 드롭다운
- Gemini Google Search grounding 기반 실제 사례 구조 참고
- 템플릿 fallback 제거
- 제목/질문 유사도 중복 제거
- 주제 추천 → 로드맵 → 실제사례 비교 → 보고서 생성
